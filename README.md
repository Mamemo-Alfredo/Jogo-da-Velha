# ❌⭕ Jogo da Velha (Tic-Tac-Toe)

Aplicação web do clássico **Jogo da Velha**, desenvolvida com **HTML, CSS e JavaScript**, com foco em lógica de programação, manipulação do DOM e interatividade.

---

## 📌 Sobre o Projeto

Este projeto implementa o jogo da velha para dois jogadores no mesmo dispositivo, permitindo alternância de turnos e verificação automática de vitória ou empate.

A aplicação foi desenvolvida com uma interface simples, responsiva e de fácil utilização.

---

## 🚀 Tecnologias Utilizadas

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript (Vanilla JS)

---

## 🎮 Funcionalidades

* 👥 Modo para dois jogadores
* 🔄 Alternância automática de turnos (X e O)
* 🏆 Detecção de vitória
* 🤝 Detecção de empate
* 🔁 Reinício da partida
* 📱 Layout responsivo

---

## ⚙️ Como Funciona

* O jogo é renderizado em uma grade 3x3
* Cada jogador marca sua jogada ao clicar em uma célula
* O sistema verifica combinações vencedoras após cada jogada
* Ao final, é exibido o resultado (vitória ou empate)
